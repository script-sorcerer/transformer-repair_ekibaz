# Локальные медиа выполненных работ

Размещайте фотографии и видео по каталогам работ:

`static/media/works/<work-slug>/<file>`

После добавления файла зарегистрируйте его в поле `gallery` соответствующей страницы в
`src/lib/seo/content.ts`. Не добавляйте запись до появления файла: пустая галерея скрывается
автоматически.
